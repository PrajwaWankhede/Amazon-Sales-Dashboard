## Project Overview

This project is an interactive **Amazon Sales Dashboard** developed using **Microsoft Power BI** and **Microsoft Excel**. The dashboard provides detailed insights into sales performance, profitability, product categories, customer behavior, and regional trends, helping stakeholders make data-driven business decisions.

## Objectives

* Analyze overall sales performance.
* Track revenue and profit trends.
* Identify top-performing products and categories.
* Monitor regional sales distribution.
* Evaluate customer purchasing behavior.
* Support strategic business decision-making.
